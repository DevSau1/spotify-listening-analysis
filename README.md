# Spotify Listening Behavior Analysis

## Overview
This project analyzes over 33,000 Spotify listening records to uncover patterns in listening behavior using Python and Tableau.

The goal was to better understand how listening habits vary by time of day and engagement level, and to apply machine learning techniques to segment user behavior.

---

## Tools & Technologies
- Python (Pandas, Scikit-learn)
- Tableau
- Google Colab

---

## Process
- Cleaned and transformed raw Spotify streaming data  
- Engineered time-based features (hour, month, day of week)  
- Performed exploratory data analysis to identify listening trends  
- Applied KMeans clustering to segment listening behavior  
- Built interactive visualizations in Tableau  

---

## Key Insights
- Listening activity peaks during evening hours (5–9 PM)  
- Three distinct listening behaviors emerged:
  - Quick Skips (low engagement)
  - Late Night Listening
  - Full Listening Sessions (high engagement)  
- Evening listening includes both high and low engagement, indicating varied user intent  

---

## Business Implications
- Low engagement periods (quick skips) present opportunities for music discovery and recommendations  
- High engagement periods (full sessions) create opportunities for increased ad exposure and monetization  
- Behavioral segmentation can be used to personalize user experience and improve retention
